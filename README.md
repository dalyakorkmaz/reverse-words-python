# Reverse words in a sentence (Python)

## Description
This project reverses each word in a sentence while preserving the word order.

## Example
Input:
hello world
Output:
olleh dlrow

## Features
- Splits sentence into words
- Reverses each word individually
- Preserves original word order

## Technologies
-Python

## How to run
python reverse_words.py

## Purpose
This project demonstrates
- string manipulation
- loops and slicing
